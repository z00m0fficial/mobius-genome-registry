# Mobius Genome Registry

## Description
Industry genome registry for Mobius. Defines the starting DNA blueprint, department archetypes, default workflows, dashboard modules, memory categories, and intelligence patterns for each organization type.

## Purpose
Every organization begins with an industry genome, then evolves into a unique Atlas DNA fingerprint based on its own verified knowledge, people, workflows, and decisions.

## Core Responsibilities
- Maintain industry genome definitions.
- Provide default department blueprints.
- Map industry type to dashboard modules.
- Support genome mutation rules as organizations evolve.
- Preserve the distinction between industry template and organization-specific DNA.

## Connected Systems
- mobius-atlas-dna
- mobius-discovery-engine
- mobius-dynamic-organization-builder
- mobius-command-center
